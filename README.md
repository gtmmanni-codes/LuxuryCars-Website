# LuxuryCars-Website

A modern luxury car dealership website built using HTML, CSS, and JavaScript. This project showcases premium sports and luxury vehicles with a clean and responsive user interface.

## Features

- Responsive Homepage
- Luxury Car Showcase
- Product Listing Page
- Car Gallery Section
- About Us Page
- Contact Form
- Modern UI Design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Pages Included

- Home
- About
- Cars
- Gallery
- Contact

## Project Purpose

This project was created as a frontend web development project inspired by real-world luxury automobile dealership websites.
